# Spotify Music Analysis
**Dataset:** 1,14,000 Spotify tracks | 125 genres
**Tool:** MS Excel (Pivot Tables, Pivot Charts, Slicers, Statistical Analysis)

## Problem Statement
Clean and Analyse Spotify music data to identify music consumption trends and generate 
insights on listener behaviour.

## Steps Performed
- Data cleaning and preparation in Excel
- Statistical analysis — correlation between danceability, energy, 
  valence, and popularity metrics across genres
- Built interactive dashboard with pivot charts and slicers for 
  dynamic genre-level filtering

## Key Insights
- Correlations identified between danceability, energy, and popularity
- Genre-level filtering reveals distinct listener behaviour patterns
- Structured report delivered with key findings and recommendations

## Output
Interactive Excel dashboard enabling dynamic exploration across 
1,14,000 tracks and 125 genres
